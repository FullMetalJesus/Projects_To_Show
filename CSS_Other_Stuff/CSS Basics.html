CSS - The language of the Web


This is a little of the knowledge I've learned about CSS in this module.

Chapter 1: Introducing CSS
  • CSS separates content from layout
  • CSS was developed to create cleaner HTML
  • CSS makes Web sites easier to maintain

CSS made up of rules that are applied to the content of a Web document. Rules consist of a selector and a declaration.

There are three ways that we can include CSS within our Web documents. They are:
  1. Inline style
  2. Embedded style
  3. External style sheets
Chapter 2: Using Basic Selectors

Selectors represent the HTML element that you want to style. Selectors can be:
  • universal selector
  • element selectors
  • class selectors
  • identification selectors
  • link selectors (grouping)
  • descendant selectors
  • attribute selectors
Chapter 3: Specifying Sizes, Colors, and Files

There are a number of ways that you can effect the size of content. Size can be:
  • Integers
  • Numbers
  • Length
  • Percentage
  • Angle (CSS3)
  • Time (CSS3)
