This is an Accordian

This is useful for displaying potentially large amounts of content in a confined space.

Extra Information

In many cases, accordians may be used to as navigational aids outside of a link drop down menu or menu tabs.

About My Prefernces

I prefer using accordians with a 'mouseover' event in order to activate them either on purpose or by accident. I find this to be a useful practice for when the end user is a novice with computers. The reason behind this is because accordians my appear to be an un-interactable/static item on a webpage to un-familiar user who may not know or think to click on the accordian.


Another feature I enjoy using with jQuery UI accordians is the 'collapsible' feature. This feature makes it possible to have all accordians deslected, in a collapsed manner, which gives more room/space to the rest of the content on the web page.
